Περιγραφή

Μια στατική ιστοσελίδα αφιερωμένη στην πόλη της Αθήνας. Το project υλοποιήθηκε στα πλαίσια της εκπαίδευσής μου στη Δ.ΣΑΕΚ Καλαμακίου για την πρακτική εφαρμογή των γνώσεων μου σε Web Technologies[cite: 2].
Τεχνολογίες που χρησιμοποιήθηκαν:

    HTML5: Για τη δομή της ιστοσελίδας

    CSS3, Javascript: Για τον σχεδιασμό και τη διαμόρφωση
